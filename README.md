# ⚡ ElectroHub – MERN E-Commerce Store

ElectroHub is a full-stack MERN e-commerce web application built using MongoDB, Express.js, React.js, and Node.js.

The project includes authentication, cart management, checkout flow, product pages, persistent login/cart storage, and a modern responsive UI.

## 📌 Features

✅ User Registration & Login  
✅ Persistent Authentication  
✅ Product Listing  
✅ Product Details Page  
✅ Add to Cart  
✅ Quantity Management  
✅ Remove from Cart  
✅ Cart Persistence using localStorage  
✅ Checkout Flow  
✅ Order Summary  
✅ Search Products  
✅ Responsive Modern UI  
✅ MongoDB Database Integration  
✅ REST API Backend  
✅ Fully Deployed Application
